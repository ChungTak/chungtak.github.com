Can't Locate Android SDK in Titanium Studio

Sep 11th, 2011

在Titanium Studio中设置了Android sdk路径，但是运行的时候提示”Can’t locate Android SDK” 原因是Titanium sdk 对 Android sdk版本有要求: + Titaniun sdk 1.7 要求至少要Android sdk 2.1 + Titaniun sdk 1.8 要求至少要Android sdk 2.2
