  • rake deploy不能发布，但是预览正常。检查github上source分支代码已更新，但master仍为老代码。

rake操作应该在source分支下进行，若是刚从github里clone下来的，请先执行$ git checkout source

命令行执行：$ git remote add octopress https://github.com/imathis/octopress.git
